About Me

Currently I'm a junior at the University of Connecticut pursuing a degree in Digital Media and Design. My focus is in Web Development and Front End Engineering. I first started building websites in high school, where I used Adobe Dreamweaver to make my first creations. That is where I discovered my love for digital media and especially websites.

Today I code all my websites by hand and use my skills and knowledge to produce content. I’m constantly striving to improve my skills in all aspects of the web. Therefore I have recently started pursuing more computer science and MIS classes in order to branch out and learn as much as possible to aid me in web development.

When I’m not designing websites, I like to read books (currently The Count of Monte Cristo), work on my investment accounts or workout at the gym.

UConn Database Project

This is a project that was initiated by me and a fellow co-worker upon seeing that the Facilities IT directions were using a huge spreadsheet in order to keep track of all the open positions as well as employees. We build a web app using mySQL and PHP for the backend and HTML, CSS, Bootstrap, and Javascript in order to create the front end of the website. It made tracking all the changes much easier and provides up to date info on all the employees. In v2 there will be the ability to assign and track custodians all over UConn.

CSS Zen Garden

This was a project done for the CSS Zen Garden website.
The goal was to take only the html provided and design a unique experience for the user using only CSS. I decided to make a blackboard themed site with text that looked like it was written on the blackboard and tilted the text in order to make it more realistic. While it may look crooked the website is still offers a great user experience.
